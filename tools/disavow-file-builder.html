Disavow File Builder

Paste a list of toxic domains or URLs and instantly generate a Google-compliant disavow.txt file. Automatic prefix detection, duplicate removal, comment support, and one-click download. Runs 100% in your browser — nothing is uploaded.

    How to upload your disavow file

    1. Sign in to Google Search Console Disavow Tool with the property owner account.
    2. Select the property.
    3. Click Replace or Submit and upload the file you downloaded here.
    4. Wait. Google typically reflects disavow within 1–4 weeks during normal recrawl cycles.
    Tip: Always keep a copy of your previous disavow file. Search Console only stores the latest version, so you lose history every time you replace it.
    Reminder: Disavow is a last-resort tool. Use only when you have a confirmed manual action or a clear ranking decline tied to a specific link pattern. Read our 30-minute audit guide first to make sure disavow is the right move.

    Frequently Asked Questions

    What is a disavow file?

    A disavow file is a plain text file you submit to Google Search Console listing domains or URLs whose links to your site you want Google to ignore. It tells Google not to count those links when assessing your site.

    When should I use disavow?

    Only when you see a clear pattern of low-quality links plus a confirmed manual action or a ranking decline that maps to that pattern. Disavowing healthy links can hurt rankings, so use it sparingly and only as a last resort.

    What is the difference between domain: and url: prefixes?

    domain:example.com disavows every link from any URL on example.com and its subdomains. url:https://example.com/page disavows only that one specific URL. Use domain: when an entire site is spammy, url: when only a few pages are problematic.

    Is my disavow data sent to a server?

    No. This tool runs entirely in your browser. Nothing is uploaded, logged, or stored. You can paste a list and download the file without any network calls.

    What is the maximum disavow file size?

    Google’s documented limit is 100,000 lines and 2MB per disavow file. This tool warns you when the output approaches those limits.
