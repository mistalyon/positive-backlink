Frequently Asked Questions

What is JSON-LD?

JSON-LD (JavaScript Object Notation for Linked Data) is the structured data format Google recommends for marking up pages. It is embedded in a script tag with type application/ld+json.

Which schema types does this validator support?

The validator checks required and recommended properties for Article, BlogPosting, NewsArticle, FAQPage, Product, Organization, LocalBusiness, BreadcrumbList, WebSite, SoftwareApplication, Person, Event and Recipe.

Does this replace Google Rich Results Test?

No. This validator gives instant offline field-level feedback while drafting schema. Always run the final markup through Google Rich Results Test before publishing.

Is my pasted data sent to a server?

No. Validation runs entirely in your browser using JavaScript. Nothing is uploaded, logged or stored.

What is the difference between required and recommended properties?

Required properties must be present for the markup to be eligible for rich results. Recommended properties improve eligibility and the quality of the rich result preview.
